## Tech Stack

- Typescript
- Nest.js
- Typeorm
- MySQL

## Getting Started

### Prerequisites

- Install [Node.js](https://nodejs.org/).
- Refer to the `.mock.env` file and create the `.env` file in the root directory.

### Run

```bash
yarn install
yarn vuild
yarn start:prod
```
